ABC LEARNING CENTER

  • Category: Static Website

  • Project date: 15 October, 2022

I created the ABC Learning Center website using HTML, CSS, and JavaScript. It is a static website featuring a homepage with comprehensive navigation for all its webpages. Each course page provides a course description and a list of included lessons.

I played a crucial role in creating an attractive and user-friendly website for the ABC Learning Center. This involved designing dynamic lists of lessons and informative course materials on each page. Additionally, I implemented an administration page, allowing the administrator to manage user registration data securely. Throughout the development, I actively sought feedback from instructors and peers, which contributed to enhancing the overall user experience. With my technical expertise and attention to detail, I successfully developed a visually pleasing static website that is functional and visually appealing for the ABC Learning Center.
